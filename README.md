# FreeLines

A native Android implementation of the classic Color Lines game, built with Kotlin and Jetpack Compose.

This project is a personalized, ad-free version of the game, designed with a clean and adaptable architecture.

## Features

- **Game Board:** A 9x9 grid.
- **Game Logic:** Move balls to form lines of 5 or more of the same color.
- **Pathfinding:** Balls can only move if a clear path is available.
- **Scoring:** Earn points for creating lines.
- **UI:** Built with modern Jetpack Compose.
